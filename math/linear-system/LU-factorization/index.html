    LU分解

    目标

    • 理解LU分解的过程

    LU分解表达

    高斯消元中，\(A\)和\(b\)矩阵是同时变化是一个缺陷，因为右手边的\(b\)也在变化。在一些应用中，希望对矩阵转换的中间结果也有所保存，LU分解就是这样一种方法。LU分解的意思是，对于\(Ax=b\)，将\(A\)分解为\(A=LU\)，其中\(L\)是下三角矩阵 (lower triangular)，\(U\)是上三角矩阵 (upper triangular)。于是，\(L\cdot y=b\)，然后\(U\cdot x = y\)，按顺序求解。

    \(A=LU\)的过程和高斯消元中的前向消元相似。LU分解的好处是，它独立于解向量\(x\)可以单独进行，此时的\(b\)向量也不需要参与其中。分解完成后，对\(L\cdot y=b\)和\(U\cdot x = y\)的求解只需要一个循环。因子分解的方法有杜利特尔分解法 (Doolittle ) 、克劳特分解法 (Crout) 、Cholesky分解法等。以下介绍Doolittle方法。分解的目标是：

    \(L=\begin{bmatrix}&1&&&&&0&\\&\alpha_{21}&1&&&&&\\&\vdots&\ddots&\ddots&&&&\\&\alpha_{i1}&\cdots&\alpha_{i, i-1}&1&&&\\&\vdots&&&\ddots&\ddots&&\\&\alpha_{n1}&\cdots&\cdots&\cdots&\alpha_{n,n-1}&1&\end{bmatrix}\)

    \(U=\begin{bmatrix}&\beta_{11}&\beta_{12}&\cdots&\beta_{1j}&\cdots&\beta_{1n}&\\&&\beta_{22}&&\vdots&&\vdots&\\&&&\ddots&\vdots&&\vdots&\\&&&&\beta_{jj}&&\vdots&\\&&&&&\ddots&\vdots&\\&0&&&&&\beta_{nn}&\\\end{bmatrix}\)

    因为\(LU=A\)，可以得到\(A\)的元素\(a_{ij}\)的表达为：

    \(a_{ij}=\begin{cases}\displaystyle\sum_{k=1}^{i}\alpha_{ik}\beta_{kj}=\sum_{k=1}^{i-1} \alpha_{ik}\beta_{kj}+\alpha_{ii}\beta_{ij}=\sum_{k=1}^{i-1} \alpha_{ik}\beta_{kj}+\beta_{ij},\quad i\leq j,\ A的上三角(含对角线)\\\displaystyle\sum_{k=1}^{j}\alpha_{ik}\beta_{kj}=\sum_{k=1}^{j-1}\alpha_{ik}\beta_{kj}+\alpha_{ij}\beta_{jj},\quad i>j,\ A的下三角\end{cases}\)

    计算过程

    从上式可以得到\(\beta_{ij}\)和\(\alpha_{ij}\)的表达式：

    \(\begin{aligned}\beta_{ij}&=a_{ij}-\displaystyle\sum_{k=1}^{i-1}\alpha_{ik}\beta_{kj},\quad i=1,2,...,j\quad j=1,2,...,n\\\alpha_{ij}&=\displaystyle\frac{1}{\beta_{jj}}\left[a_{ij}-\sum_{k=1}^{j-1}\alpha_{ik}\beta_{kj} \right],\quad i=j+1,j+2,...,n \end{aligned}\)

    上式的解释：\(\beta_{ij}\)的表达式限制为行数\(i\leq j\)，为上三角，所以\(i=1,2,...,j\)，列数\(j\)则可以从头取到尾，\(\alpha_{ij}\)的表达式是给下三角的，所以行数\(i>j\)。而且，可以发现，每一个\(A\)中的\(a_{ij}\)只会决定一个\(\alpha_{ij},\ i>j\)或者一个\(\beta_{ij},\ i\leq j\)。这样一个好的性质决定了可以用单个数列 (single array) 来实现LU分解。

    直观来看：

    \(LU=\begin{bmatrix}&a_{11}=\beta_{11}&a_{12}=\beta_{12}&\cdots&&&a_{1n}=\beta_{1n}&\\&a_{21}=\alpha_{21}\beta_{11}&a_{22}=\alpha_{21}\beta_{12}+\beta_{22}&\cdots&&&a_{2n}=\alpha_{21}\beta_{1n}+\beta_{2n}&\\&\vdots&\vdots&&&&\vdots&\\&a_{i1}=\alpha_{i1}\beta_{11}&a_{i2}=\alpha_{i1}\beta_{12}+\alpha_{i2}\beta_{22}&\cdots&&&a_{in}=\alpha_{i1}\beta_{1n}+\alpha_{i2}\beta_{2n}+...+\beta_{in}&\\&\vdots&\vdots&&&&\vdots&\\&a_{n1}=\alpha_{n1}\beta_{11}&a_{n2}=\alpha_{n1}\beta_{12}+\alpha_{n2}\beta_{22}&\cdots&&&a_{nn}=\alpha_{n1}\beta_{1n}+\alpha_{n2}\beta{2n}+...+\beta_{nn}&\end{bmatrix}\)

    于是，一个稳健有效的方法是遍历列，因为：

    • 每一列的\(\beta_{1j}\)都可以直接得到
    • 每一列出了\(\beta_{1j}\)之外的其它\(\beta\)，都可以从上面的\(\beta\)和左边的\(\alpha\)得到
    • 每一列的\(\alpha_{ij}\)都可以从上面的\(\beta\)和左边的\(\alpha\)得到

    简单来说，举个例子，为了得到\(\beta_{22}\)，必须知道\(\alpha_{21}\)，所以先求得第一列。

    计算过程，首先计算出第\(j\)列的每个元素\(\beta_{ij},\ i=1,...,j-1\)，然后计算\(\tilde \alpha_{ij},\ i=j,...,n\)，最后通过除以\(\beta_{jj}\)得到\(\alpha_{ij}\)：

    \(\begin{aligned}\beta_{ij}&=a_{ij}-\displaystyle\sum_{k=1}^{i-1}\alpha_{ik}\beta_{kj},\quad i\leq j\\\tilde \alpha_{ij}&=a_{ij}-\displaystyle\sum_{k=1}^{j-1}\alpha_{ik}\beta_{kj},\quad i=j,j+1,...,n\\\alpha_{ij}&=\tilde \alpha_{ij}/\beta_{jj},\quad i=j+1,...,n \end{aligned}\)

    PA=LU分解

    带行交换的LU分解，涉及到置换矩阵 (permutation matrix) 和 偏置主元 (partial pivoting) 的概念。

    置换矩阵：每一行有一个1，其他都是0

    偏置主元：为每一行做高斯消元之前，先在主元所在列找到绝对值最大的数，将该数所在行换到消元行。
