    线性系统求解

    目标

    • 了解求解\(Ax=b\)的方法一览

    概览

    线性系统经常出现在两个方面：1）描述一个问题的数学表达；2）应用特定线性逼近方法解决问题而出现。通过有限差分法处理的微分方程边界值问题（BVP）的离散化，或者光滑方程的线性插值是两个典型的会出现线性系统的方法。

    求解线性系统可以粗略地分为两种方法：

    • 直接法（direct methods）
    • 迭代法（indirect methods）

    直接法是有限算法：包含解的元素操作数量，直接取决于线性系统的大小。直接法只会受到计算时四舍五入误差的影响（round-off errors），这个特性让它在当线性系统大小/元素操作的数量不是很极端的情况下很受欢迎。然而，尽管直接法的概念很简单，对操作实施克莱姆法则（Cramer's rule）却不是很有效，所以不是很推荐在现实世界的应用中使用。直接法的常见例子有：

    • 高斯消元 (Gaussian elimination)
    • 高斯-若尔当消元 (Gaussian-Jordan elimination)
    • LU分解 (LU factorization)
    • Cholesky分解 (Cholesky decomposition)

    迭代法，数学表达上是可以无限迭代逼近的，但是现实中无法做到无限迭代。它受到四舍五入误差的影响，同时也因为有限次迭代而受到截断误差 (truncation errors) 的影响。它的使用简单直接，在使用直接法无法进行追溯的情况下，非常适合使用。此外，迭代法可以用于完善其他方法获得的解。迭代法的常见例子有：

    • 雅可比迭代 (Jacobi)
    • 高斯-塞德尔迭代 (Gaussian-Seidel)
    • 松弛法 (relaxtion method)

    1 - 直接求解

    1.1 - 求逆

    \(x=A^{-1}b\)

    注意事项：

    • 维度比如< 100还行
    • 复杂度\(O(n^3)\)，矩阵太大不行

    1.2 - 因子分解

    对于\(A\)矩阵，有：

    \(A=\begin{pmatrix}&LU, A\ is \ square\ matrix\\&LDL^T,if\ A=A^T\\&LL^T,if\ A=A^T\ and\ A>0\end{pmatrix}\)

    A为方阵

    则可以分解为上三角和下三角

    A为对称阵

    可以对角阵分解

    A为对称阵且正定

    可以分解为三角形矩阵和它转秩的矩阵

    求解\(LL^T x=b\)的过程分为两部，第一步是令\(y=L^T x\)，求\(Ly=b\)中的\(y\)，第二步是求\(L^T x=y\)中的\(x\)

    注意：求解对角阵的线性系统时，求解过程是前后依赖的，无法使用GPU（或者在使用GPU的程序中运行）

    使用：在建立某个程序的原型时可以使用

    注意事项

    • 时间复杂度
      • 如果A是稠密矩阵，分解的时间复杂度也是\(O(n^3)\)
    • 所以，当A为稀疏矩阵的时候，该方法比求逆要好一些
      • 维度撑死不能超过100万

    2. 迭代求解

    2.1 基于不动点

    基于不动点的的迭代线性求解器 Stationary iterative linear solvers，求解的过程是平滑的方式

    • 高斯消元，复杂度\(O(n^3)\)
    • 雅可比迭代 Jacobi ，复杂度\(O(n^2)\)
    • 高斯塞德尔迭代 Gauss-Seidel iterations \(O(n^2)\)
    • SOR
    • Multigrid

    2.2 Krylov subspace方法

    • 共轭梯度 Conjugate gradient (CG)

      • 当矩阵正定且对称时：\(A=A^T\)并且\(A>0\)

      • 保证\(n\)次迭代一定能收敛

      • 求解效率依赖于一个kappa值：\(\kappa=\displaystyle\frac{\lambda_{max}}{\lambda_{min}}\)

        • \(\lambda\)是\(A\)矩阵的特征值，即最大特征值和最小特征值的比值

        • 如果\(\kappa\)很小，例如等于1的时候，共轭梯度非常非常非常好用

        • 只需要二十行代码就可以搞定

    • biCG

    • CR

    • MinRes

    • GMRes

    3 - 稀疏矩阵

    如果矩阵中的\(n^2\)个元素只有\(n\)个是非零的。..

    高斯消元和\(PA=LU\)分解不适合稀疏矩阵，所以需要迭代法。

    但是，对于稀疏矩阵，迭代法也有局限，例如一个矩阵有一百万行\(n=1.000.000\)，即\(n\times n=(1e6, 1e6)\)，那么需要储存\(n^2=10^{10}\)个浮点数，需要\(8\times 10^{10}\)个字节，等于80G储存空间。迭代的时间也不行...

    稀疏矩阵的储存可以使用三元数\((i,j,d)\)，其中\(d\)是实数，也就是只储存需要计算的。

    对称正定矩阵

    获得对称正定矩阵 (Symmetric positive-defined matrix) 的方法是Cholesky分解。如果\(A\)是对称正定的，\(Ax=b\)的解法也不一样。

    共轭梯度非常适合求解大型稀疏矩阵。

    每个对称正定矩阵\(A\)都可以表达为\(A=R^TR\)，\(R\)为上三角矩阵。

    共轭梯度是预处理 (preconditioning) 的入门简介。

    \(A\)是对称切正定的，那么\(A^T=A\)，并且\(A>0\)。对于非零向量\(x\)，可以得到\(x^TAx>0\)

    4. 总结

    当求解的规模相当大，例如\(1e^6\)或者更多，比如求解1亿个粒子时，只能使用迭代求解器
